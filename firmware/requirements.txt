# Deks 1.0 ESP32 펌웨어 요구사항
# 마이크로파이썬 기반 ESP32 S3 펌웨어

# 마이크로파이썬 버전
# ESP32-S3용 마이크로파이썬 v1.21.0 이상 권장

# 하드웨어 요구사항
# - ESP32 S3 DevKitC-1
# - FIT0405 DC 기어드 모터 2개 (엔코더 내장)
# - L298N 모터 드라이버
# - 적외선 거리센서 2개
# - 1588AS LED 매트릭스 (8x8)
# - 5V 패시브 버저
# - 3.7V 리튬 폴리머 배터리 (2000mAh)
# - TP4056 충전 모듈
# - 전압 레귤레이터 (3.3V, 5V)

# 소프트웨어 요구사항
# - 마이크로파이썬 펌웨어 (ESP32-S3용)
# - esptool.py (펌웨어 업로드용)
# - ampy 또는 rshell (파일 업로드용)

# 네트워크 요구사항
# - Wi-Fi 네트워크 접근
# - 서버 IP 주소 및 포트 설정
# - TCP 소켓 통신 지원

# 개발 도구
# - Python 3.7+ (개발 환경)
# - esptool.py
# - ampy 또는 rshell
# - 시리얼 터미널 (115200 bps)

# 테스트 도구
# - 멀티미터 (전압/전류 측정)
# - 오실로스코프 (PWM 신호 확인)
# - 전원 공급기 (개발 중 전원 공급)
